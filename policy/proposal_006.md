# 提言006：NHKの解体・分社化による受信料大幅引き下げと国家映像戦略への転換

---

## 0. 問題意識

現在のNHKは「法的義務」を盾に、国民から集めた高額な受信料であぐらをかいている。肥大化した組織、年功序列で平均1,100万円に達する職員給与、不透明な研究開発費、そして義務があるにもかかわらず徴収業務に年500億円超を費やす構造的な矛盾。これらはいずれも、競争にさらされていない組織が必然的に辿り着く末路である。

Netflixが月額890円で世界規模のコンテンツを提供し、民間放送局が広告収入だけで運営を続けている。法的義務に守られた公共放送が持つべきは、その義務に見合ったコスト意識と、義務に見合った業務範囲だけである。

本提言は、NHKを純粋な報道・教育・災害インフラに徹底的にスクラップ＆ビルドし、受信料を現在の約3分の1（**月額600円**）まで引き下げることを提言する。

## 2. 改革の4つの柱

---

### 柱① 組織の分社化とNHK本体の公共特化

民放と競うようなバラエティ番組、商業的なエンタメ、過剰なスポーツ放映権を完全廃止。NHK本体が担う業務を、国民全員が費用負担する根拠のある領域だけに絞り込む。

**廃止する業務**
- バラエティ・クイズ・トーク・商業ドラマ
- 民間代替可能なスポーツ中継
- NHKオンデマンドの商業的コンテンツ
- 過剰な国際展開（NHKワールドの一部）

**NHK本体が担う業務（公共インフラとして存続）**
- 国内・国際ニュース報道
- 災害報道・緊急警報システム
- 選挙特番・国会中継
- 教育番組（Eテレの核心部分）
- 気象情報の公共放送

**文化的価値として維持するコンテンツ（2つのみ）**

| コンテンツ | 維持の根拠 | 条件 |
|---|---|---|
| **紅白歌合戦** | 日本の音楽文化・歴史の継承 | 「純国産」構成に原点回帰。視聴率・海外タレント依存の演出を廃止し、制作費を大幅削減 |
| **大河ドラマ** | 歴史教育・文化的アーカイブとしての公共性 | 教育的観点を軸に制作規模を適正化 |

廃止されたエンタメ部門は分社化または民間への売却・移管とし、NHK本体の予算から完全に切り離す。

---

### 柱② 人件費を「テレビ業界の中央値」へ連動・適正化

年功序列で自動的に平均1,100万円に達する給与体系を廃止する。地方の小規模テレビ局（山口・青森・岩手等）で働く人々を含む、日本全国のテレビ業界全体の中央値（約700〜750万円）とNHK職員の平均給与が完全に一致するよう、**放送法または省令で上限を義務化**する。

```
NHK職員の平均給与 = 全国テレビ業界の全職種平均給与（中央値）
                   ≈ 約700万〜750万円
```

毎年の厚生労働省「賃金構造基本統計調査」の放送業区分データに自動連動させ、裁量の入る余地をなくす。

- 現行平均1,100万円 → 目標700〜750万円
- 分社化・自然減による段階的実施
- **削減効果：約350〜450億円／年**

---

### 柱③ 技研（放送技術研究所）の廃止と国立大学への完全移管

NHK放送技術研究所は、年間約250〜280億円の受信料を投じて次世代映像技術・空間コンピューティング・AIアーカイブ等を研究しているが、その成果はNHK内部に囲い込まれ、民間放送局や配信事業者には開放されない。

「国民全員が負担する受信料」で生まれた技術が、民間に非公開である。この構造は根本的に正当化できない。

**移管の設計**

| 移管先 | 対象領域 | 財源 |
|---|---|---|
| トップ国立大学3〜4校 | 次世代映像技術・空間コンピューティング・AI映像処理 | 科学技術振興費（国費）＋大学運営費 |

**最大のポイント：オープン化**

移管後の研究成果は、日本のすべての民間放送局・配信事業者・クリエイターへ**オープンライセンスで提供**する。NHKが囲い込んでいた技術インフラを解放することで、日本のコンテンツ産業全体がNetflixや海外メディアと対等に戦えるプラットフォームを国主導で整備する。

- **削減効果：約250億円**（受信料からの支出がゼロになる。国費への付け替えは科学技術予算の組み替えで対応）

---

### 柱④ 営業費の完全撤廃と「インフラ決済」による一括徴収

法的義務として支払い義務が存在するにもかかわらず、NHKは年間500〜600億円を督促・訪問・郵送・コールセンター・委託業者に費やしている。気象庁・消防庁・国立国会図書館は「義務だから徴収業務に予算をかける」という発想を持たない。NHKだけが例外である理由はない。

**新しい徴収設計**

```
Option A：電気料金への上乗せ（既存インフラ活用）
Option B：マイナンバー紐づけによる住民税・社会保険料との一元徴収
```

どちらも、NHK独自の徴収業務はゼロになる。

**「複数住居二重払い」の完全廃止**

現行の「住居単位の契約」を廃止し、1世帯＝1つの「デジタル公共アカウント」へ一本化する。単身赴任・学生の別居・セカンドハウス等での二重払いを撤廃し、スマートフォン・PCでの視聴も追加料金なしで同一アカウントから利用できる。

- **削減効果：約500億円**

| 人件費の業界中央値連動 | ▲450億円 |

| **削減総額** | **▲約2,500億円** |

```

削減総額        ▲約2,500億円

```

## 4. 「国費化」の大義名分：なぜ今なら可能か

これまで受信料の「税連動徴収・国費化」は、一つの懸念によって繰り返し阻まれてきた。

> **「時の政権が予算を握ることで、報道の独立性が損なわれる」**

この懸念は正当である。国営放送化は報道の自由の観点から絶対に避けなければならない。

しかし、本提言が設計するNHKは構造的に異なる。

```
現在のNHK：エンタメ・スポーツ・研究・商業サービスを抱えた「巨大メディア企業」
               ↓
提言006後のNHK：ニュース・災害・教育に特化した「報道インフラ」
```

純粋な報道・災害インフラに特化したNHKは、気象庁・消防庁・国立国会図書館と同じカテゴリに属する。これらの機関は国費で運営されているが、「政権が内容を支配している」とは誰も言わない。**インフラの維持費と、報道内容の編集権は、切り離せる**からである。

本提言が求めるのは「NHKを政府の広報機関にせよ」ではない。「肥大化したエンタメ部門を切り離した上で、残った純粋な報道インフラの**維持費**を、受信料という非効率な私的徴収から、税という効率的な公的調達へ移行せよ」である。

**スリム化して初めて、国費化の大義名分が立つ。この順序が本提言の核心である。**

## 5. 前提条件・今後の検討課題

- 分社化された「エンタメNHK」の株式・資産の帰属と処分方法
- 技研の人材・知的財産（特許等）の大学移管における法的整理
- 電気料金上乗せ vs マイナンバー一元徴収の選択における公平性の検証
- 「デジタル公共アカウント」の設計・セキュリティ・運用主体
- 受信料廃止後の放送法改正における「編集権の独立」の明文化
- BS廃止統合に伴う衛星帯域の利用・返還ルール
- 移行期間（既存職員・契約・番組の段階的整理）の設計
