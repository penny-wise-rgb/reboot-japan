# 【提言003】BtoB下請け産業の構造再編と成長ドライブ施策
## ～ 零細企業を減らし、強い中堅企業を育てるための産業大手術 ～

---

## 0. 提言001との接続

本提言は、**提言001「日本再起動（REBOOT）プラン」と一体で機能する**。

提言001が設計した「最低賃金2,000円」「失業給付3年＋再教育」「フレキシキュリティ型セーフティネット」は、本提言によって発生するゾンビ企業の退場と雇用流動化を支える受け皿である。

```
提言003：企業の新陳代謝を強制的に促す（産業側の設計）
提言001：退場した企業の労働者を守り、再教育する（労働者側の設計）

この２つは車の両輪である。
```

## 1. 本提言の基本理念と目的

日本の労働者の約7割は中小・零細企業に勤めているが、過去30年間、彼らの賃金はほとんど上がっていない。その元凶は、無条件の優遇税制と目的を問わない補助金が市場の新陳代謝を阻害し、自力で成長できないゾンビ企業や多重下請けの中抜き業者を延命させてきたことにある。

**現在の企業規模別分布（企業数ベース・2021年経済センサス）**

| 区分 | 企業数 | 全体比 |
|---|---|---|
| 大企業 | 約1万社 | 約0.3% |
| 中堅企業（従業員2,000人以下） | 約9,229社 | 約0.7% |
| 中小企業 | 約213万社 | 約58% |
| 零細・小規模事業者 | 約123万社 | 約41% |

さらに深刻なのは**縮小の流れ**である。自然状態では、5年間で小規模から中規模に成長した企業は3.8万社であるのに対し、逆に縮小した企業は6.5万社。放置すれば零細化は加速する一方だ。

そして最も看過できない数字がある。**全法人の61%が赤字法人**である（令和5年度・国税庁会社標本調査）。さらに令和5年度を含む**過去10年の平均でも63〜64%**が赤字法人だ。

この数字には重大な注釈がある。過去10年でこれが「改善後の水準」である。リーマン・ショック後の2010年度には76%に達しており、その後緩やかに低下してきた。しかし2020〜2021年のコロナ禍においては、給付型補助金・助成金が企業の利益を押し上げた結果として61〜62%台に見えているに過ぎない。補助金を剥がせば63〜64%に戻るのが実態と読むべきだ。

景況感による一時的な赤字は致し方ないとしても、10年平均でこの水準が続くのは異常である。節税目的で「作られた赤字」が常態化していると考えるのが自然だ。これを許容し、さらに補助金まで投じてきたのが現行制度の最大の欠陥である。

本提言は、この流れを**強制的に逆転させる**。

## 2. 対象

### 対象企業規模

**大企業を除く、中小・零細企業**に限定して実施する。大企業はすでに市場原理の中で競争しており、本提言の対象ではない。

### 対象業種

多重下請けによる「中抜き」と「低賃金」が最も深刻な以下の3業種に**100%限定**して実施する。

- **建設・土木・建築業界**：実体のないペーパーカンパニーを排除し、職人を自社雇用する筋肉質な中堅建設会社へ再編
- **IT・システム開発業界**：エンジニアを右から左へ流す「SES多重中抜き屋」を全滅させ、開発力のあるSIerへ集約
- **製造業（部品加工等）**：単独延命を諦めさせ、同業合併によって親メーカーと対等に渡り合える「強いサプライヤー」へ進化

> **【重要除外規定】文化の多様性を守るための「盾」**
>
> 飲食、美容、アパレル、エンタメなどの「対個人接客業（BtoC）」は、効率や規模だけでは測れない「情緒・多様性」そのものが付加価値であり、日本文化の豊かさそのものである。これらは本施策の対象から**完全に除外**し、現行の優遇枠組みで維持・保護する。

### BtoB/BtoC の判定基準

同一企業がBtoBとBtoCの両方を行う場合は、以下の二重基準で判定する。

```
① 売上に占めるBtoB取引が50%超の企業 → 対象
② 日本標準産業分類で建設・IT・製造業に該当する企業 → 原則対象
（①②いずれかに該当すれば対象とする）
```

## 3. 現行制度の抜本的見直し

### 無条件優遇税制の廃止

**現行の中小企業向け無条件優遇税制（軽減税率15%）は廃止する。**

法人税率は企業規模にかかわらず**一律25%（実効税率ベース）**とする。

現行の格差と新制度の比較：

| 区分 | 現行実効税率 | 新制度 |
|---|---|---|
| 中小企業（所得800万円以下） | 約21% | 25%（フラット） |
| 大企業 | 約30% | 変更なし |
| 条件達成企業（維持） | ― | 20%（5pt優遇） |
| 条件達成企業（拡大） | ― | 15%（10pt優遇） |
| 条件未達成企業 | ― | 25%（優遇なし） |

創業から**3年間のみ**、無条件で現行水準の優遇を適用する（新規創業の保護）。4年目以降は成長条件の達成が必須となる。

### 補助金の抜本的見直し

**現行の中小企業向け補助金（年間推定1〜2兆円規模）を抜本的に見直す。**

```
【廃止】持続化・経営維持・延命目的の補助金すべて
　→ 小規模事業者持続化補助金
　→ 事業再構築補助金（現行スキーム）
　→ その他、成長に紐づかない補助金全般

【存続・拡充】拡大・成長目的の補助金のみ
　→ M&A・合併支援
　→ 設備投資（雇用拡大に紐づくもの）
　→ 再教育・リスキリング支援（提言001と連動）
```

**赤字法人への補助金は原則廃止。** 補助金は「次の一手を打つための資金」であって「赤字の穴埋め」ではない。

### 赤字法人の正常化目標

現在61%の赤字法人比率を、**10年平均で10〜20%まで削減**することを目標とする。「作られた赤字」による節税を厳格に取り締まり、本当の意味での黒字企業を増やすことで、法人税の課税ベースを拡大する。

## 4. 施策パッケージの3大柱

### 柱①　二段階・成長判定システム（恒久制度）

判定は**厚生年金被保険者ベースの年間平均在籍数**で行う。「1年だけ増やして翌年クビにする」小手先のハックを防ぐためである。

**【維持条件】── 優遇税制（20%）を得るための最低ライン**

```
条件A：一人当たり賃金がインフレ率×120%以上増加
　　　　（例：インフレ3%の年 → 手取りベースで3.6%増）
条件B：正規雇用者数が前年比マイナスにならない
```

**【拡大条件】── さらに手厚い優遇（15%）を得るための上位ライン**

```
条件C：正規雇用者数が前年比3%以上増加
条件D：賃金総額が前年比5%以上増加
```

**【未達成の場合】**

```
条件を1年でも下回った場合 → 優遇を即時全廃（25%に戻る）
翌年以降に条件を再達成した場合 → 優遇を再適用
```

本制度は**恒久制度**とする。ただし10年数値目標（後述）の達成状況に応じて優遇幅を見直す。

```
目標達成の場合　：優遇幅を縮小、または新たな数値目標を設定
目標未達成の場合：優遇幅を拡大してM&Aと成長をさらに促進
```

### 柱②　M&A（合併）の完全無税化（恒久制度）

自力での成長が難しい中小・零細企業に対し、「合併して強くなる」ための圧倒的なインセンティブを恒久的に用意する。

**M&A特例の内容**

```
・合併した年：両社とも「拡大条件達成」とみなす（15%優遇）
・合併に関わる登録免許税、不動産取得税、仲介手数料への課税：ゼロ
・合併翌年以降：合算ベースで通常の維持・拡大条件を適用
```

本特例も**恒久制度**とする。「合併して強くなる」ことは特定期間だけ奨励すべき行動ではなく、産業構造の恒久的な方向性だからである。

**合併後の人員について**

日本の解雇規制（整理解雇の4要件）により、合併直後の強制的な大量解雇は法的に困難である。現実には早期退職優遇・自然減による人員調整が中心となる。「合併ゾンビ」（不要な人員を抱えたまま合併を繰り返す企業）は、維持条件Aの「インフレ率×120%の賃金増」によって自然淘汰される設計になっている。

### 柱③　BtoB経費の完全ホワイト化と元請け連座制

**経費の厳格化**

生産性に寄与しない経費（高級外車・個人的飲食の付け回し等）での利益圧縮を「脱税」とみなして厳罰化。出た利益は「設備投資」か「雇用・賃金増の原資」に回すことを法律で強制する。これが61%に及ぶ「作られた赤字」を正常化する直接の手段でもある。

**元請け連座制（段階的制裁）**

下請けがクリーン化に違反していた場合、それを放置した元請け（ゼネコン・大手SIer）にも責任を問う。公共インフラへの影響を考慮し、段階的に制裁を適用する。

```
【通常違反】　　　：法人税の加重課税（抑止力）
【重大・反復違反】：官公庁入札資格の停止（最終手段）
【災害・緊急時】　：入札停止を自動解除する例外規定を適用
```

## 5. 制度移行の設計

急激な制度変更は企業活動を混乱させる。**2年間の告知期間**を設け、各企業が準備できる環境を整える。

```
【告知開始】制度開始2年前に全内容を公表
　→ 各企業がM&A・採用・賃金設計を前倒しで検討可能

【移行期間】告知から制度開始まで
　→ 現行制度をそのまま継続（連続性の確保）
　→ 現行の優遇と新制度の優遇の間に「崖」を作らない

【制度開始年】
　→ 新制度本格スタート
　→ M&Aの集中発生は許容・むしろ歓迎
　→ 補助金の廃止も同時実施
```

## 6. 10年数値目標

```
【現状】中堅企業：9,229社（全体の0.7%）
【目標】中堅企業：約27,000社（全体の2.1%）
　　　　→ 10年で3倍増
　　　　→ 年間約1,800社ペースで中堅化

【赤字法人比率】現状61% → 目標10〜20%（10年平均）
```

年間約1,800社の中堅化は、自然状態での成長ペース（年間約7,600社が小規模→中規模に成長）に、M&A無税化・優遇条件化というインセンティブが加わることで、十分に達成可能なラインである。

**目標達成状況による制度の見直し**

```
10年後に数値目標を達成した場合
　→ 優遇幅を縮小、または新たな上位目標を設定して継続

10年後に数値目標が未達成の場合
　→ 優遇幅をさらに拡大して延長
　→ 「中堅27,000社」の達成を制度縮小の条件とする
```

## 7. 税収変化の試算

### 前提条件

```
・対象3業種の中小零細企業：約100万社
・そのうち黒字法人：約39万社（現行39%）
・条件達成率：40%（約16万社）
・インフレ率：3% → 条件A：賃金3.6%増
・対象3業種の中小零細従業員数：約700万人
・平均年収：380万円（中央値）
・所得税は現行税率ベースで計算（提言001のフラット税は含まない保守的試算）
```

### 税収変化の内訳（条件達成率40%・移行完了後）

| 項目 | 増減 |
|---|---|
| 法人税フラット化＋条件優遇による増収 | +約2,500億円 |
| 所得税増収（賃金3.6%増・280万人分・現行税率ベース） | +約3,800億円 |
| 社会保険料増収 | +約1.1兆円 |
| 消費税増収（手取り増加分の消費転換） | +約3,400億円 |
| 補助金廃止による歳出削減 | +約1〜2兆円 |
| 赤字法人の黒字化（61%→20%）による法人税増収 | +約1.5兆円 |
| **合計** | **+約4.6〜5.6兆円/年** |

### 初年度の注意点

- 軽減税率廃止による中小企業の負担増への反発が予想される
- ゾンビ企業退場による一時的な法人税収の減少が生じる
- ただし2年間の告知期間と初年度の現行制度維持により、急激なショックは緩和される
- 提言001の財源試算「初年度3兆円赤字」と同じ構造であり、3年後の黒字転換を見込んだ設計である

```
① ゾンビ経営者の自然淘汰
```

## 9. 本提言の哲学

本施策は「雇用を増やし、賃金を上げた企業をトコトン優遇する」という、**誰も表立って反対できない最強のポジティブな大義名分**を掲げている。

メディアや業界団体に「弱者いじめ」と叫ぶ隙を与えないまま、その実、中身は「成長しない企業・ズルをする企業を市場から退場させる」という、日本経済復活のための最も冷徹で機能的なロジックである。

そして退場した企業の労働者は、提言001のセーフティネットが確実に受け止める。

**強さと優しさは、矛盾しない。**
