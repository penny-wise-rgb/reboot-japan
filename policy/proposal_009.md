# 提言009：高校教育の「全面公立化」と地方財政傾斜配分による地方創生戦略

**～年間6,800億円のバラマキを回収し、地方の未来を創るコモンズ（公共財）へ投資せよ～**

---

## 0. 問題意識と大原則

高校進学率が98.8%に達した現代の日本において、高校はもはや「選別のための機関」ではない。18歳の成人を社会へ送り出す前の**「実質的な義務教育（最後のセーフティネット）」**である。

にもかかわらず、国は年間約6,800億円の公費を私立高校の維持・学費の肩代わりに投じ続けている。本提言は、この予算を全額回収し、公立高校への集中投資と地方再生の長期戦略に組み替える。

---

## 1. 提言の3つの核心（グランドデザイン）

---

### 核心① 「私立補助金」の全面廃止と私立の公立化

#### 廃止する支出

| 廃止する支出 | 金額（国費ベース） |
|---|---|
| 高等学校等就学支援金（私立・所得制限撤廃分） | 5,824億円 |
| 私立高等学校等経常費助成費（私学助成） | 1,006億円 |
| **廃止合計** | **約6,830億円** |

#### 回収した予算の最優先用途：教員給与の抜本的引き上げ

回収した6,830億円は、まず最初に**教員給与の引き上げに充当する**。教育の質は教師の質で決まる。優秀な若者が教職を真剣に選択肢に入れる水準まで給与を引き上げることで、数年後に質の高い人材が教育現場に流入する長期の連鎖をつくる。

**人に教えるという行為は、本来尊いものである。その価値を給与水準に反映させる。**

```
私立補助金廃止（6,830億円）
　↓ まず最初に優先充当
教員給与の引き上げ（約850億円）
　↓ 残りを
80/20分割（学校・インフラ）へ
```

**40歳時点での年収目標（給与所得者の所得分布基準）**

| 職種 | 目標水準 | 年収目安（40歳） | 現行比 |
|---|---|---|---|
| 高校教員 | 上位10%（90パーセンタイル） | 約1,000〜1,100万円 | 現行約650万円の約1.6倍 |
| 大学教授（標準） | 上位5〜10% | 約1,100〜1,500万円 | 現行約1,100万円→上限引き上げ |
| 世界的研究者 | 上位1%（99パーセンタイル） | 約3,000万円〜 | 国際的な人材獲得競争に対応 |

高校教員が40歳で1,000万円を超えるキャリアパスを設計することで、医師・弁護士・外資コンサルと並ぶ職業として教職が機能し始める。

**財源試算**：高校教員約23万人の平均年収を650万円→1,050万円へ引き上げた場合、追加人件費は約**850億円／年**。私立補助金廃止の6,830億円の範囲内で十分吸収できる。

#### 移行設計：廃校ではなく「公立化」

私立高校を一律廃校にするのではなく、**公立高校として転換・存続させる**。これにより「私立が消えて受け皿がなくなる」という逆説を完全に回避する。

| 条件 | 移行後の扱い |
|---|---|
| 学長が継続意思あり | 宗教色を廃止した上で、都道府県の枠内で独自性を一部維持 |
| 学長が継続意思なし | 都道府県が方針・運営を完全に引き継ぐ |
| いずれの場合も | 宗教教育は廃止。公立としての中立性を担保する |

**移行期間：最低3年**（現在の在校生が全員卒業できるまで現行制度を継続）

真に自立できる私学——宗教団体・民間企業・富裕層の授業料のみで運営できる学校——は存続を認める。

### 核心② 「過疎地傾斜配分」と予算の80/20分割設計

#### 予算フロー

```
国（6,830億円）
　↓ 教員給与引き上げ分（約850億円）を優先充当後
都道府県（受け取り・分割）
 ├→ 20%：インフラ・企業誘致専用予算（用途限定）
 │         ↓ 住民投票で使途を確定（四半期ごと）
 │         学校運営が権限を持ち執行
 │
 └→ 80%：公立高校への傾斜配分
           ↓ 学校運営が権限を持ち執行
           各公立高校が自律的に運用
```

20%は都道府県の一般財源に入れず、**専用口座（エスクロー）として分離管理**する。未執行のまま年度末を迎えた場合は、自動的に翌年の80%枠（学校分）へ戻す。

#### 傾斜配分の設計（80%部分）

| 対象 | 1校あたり年間配分 |
|---|---|
| 都市部の公立高校（生徒数多） | 1億円〜1.5億円 |
| 過疎地の公立高校（生徒数少） | **3億円〜4億円** |

「地方の公立高校のほうが、1人あたり予算が数倍・最高峰のデジタル設備・民間プロ講師によるマンツーマン並みの伴走を完全無料で受けられる」という状態を意図的につくる。

さらに、移住世帯限定の**「未来創生活動支援金（1プロジェクト最大100万円）」**を学校から直接給付する。

#### 学校予算の使途：学校運営が権限を持つ

予算の使途決定権は**学校運営（校長・運営委員会）**に委ねる。都道府県は配分のルールのみを担い、使途への介入は行わない。透明性はダッシュボードによる完全公開で担保する。

**認められる支出（例）**
- 民間プロ講師・実務家の採用
- 教育方針に連動した施設整備（体育館・グラウンド・校舎改修等）
- 最新デジタル設備・学習環境の整備
- 生徒のチャレンジ基金

**施設整備（箱物）について**

箱物投資は一律禁止しない。スポーツに特化する高校がグラウンドや体育館を整備すること、きれいな校舎や備品が「この学校に行きたい」という差別化につながることは、正当な教育投資である。

ただし**地域の既存インフラと重複する無意味な投資は禁止**する。

```
判定の2段階：
① システムによる自動照合：地域の既存施設リストと照合し、
  重複投資を自動的に弾く
② 住民投票による承認：①を通過した投資案件を
  住民投票で最終承認する
```

例：近隣に大型陸上競技場があり平時に高校が利用可能な地域への新設陸上トラック → 自動照合で弾く

#### 20%予算の使途決定：住民投票制度

インフラ・企業誘致への20%予算の具体的な使途は、**四半期ごとの住民投票**で確定する。マイナンバーとスマートフォンを活用し、行政コストをかけずにリアルタイムで民意を反映する。

**投票設計（年齢・属性別の票数）**

| 投票者 | 票数 | 根拠 |
|---|---|---|
| 中学生・高校生 | 3票 | 教育予算の最大の当事者 |
| 10〜20代 | 3票 | 恩恵を最も長く受ける世代 |
| 30〜40代 | 2票 | 子育て世代・地域の担い手 |
| 50代以上 | 1票 | 地域の経験値を反映 |
| 移住世帯（年齢問わず） | ＋1票（ボーナス） | 移住促進のインセンティブ |

通常の選挙では実現できないルール（年齢別加重投票・中高生への投票権）をこの制度で先行実験する。

#### 地方再生の連鎖モデル（10〜20年スパン）

```
教育予算の傾斜配分（過疎地に集中投資）
　↓
教員給与引き上げ → 優秀な教師が地方に来るインセンティブ
20%でインフラ整備・企業誘致（住民投票で確定）
　↓
教育環境の質が上がり、若年世帯が地方へ移住し始める
　↓（10〜20年スパン）
人が集まる → 企業が集まる → 仕事が増える
　↓
所得が上がる → 商業施設が増える → 税収が上がる
　↓
医療が整う → コミュニティが生まれる → さらに人が集まる
```

### 核心③ 教育権限の「国から都道府県・学校」への完全委譲

国の一律で硬直化した学習指導要領を廃止し、カリキュラム・学部設置の権限をすべて都道府県へ移譲する。さらに予算執行権は学校運営に委ねる。教育を「地方自治体・学校が生き残るための最大の戦略投資」に変える。

**地域産業に直結した公立「専門学部」の創設（例）**

| 地域の特性 | 創設する専門学部 |
|---|---|
| IT・DX推進地域 | AI・データサイエンス学部 |
| 医療・ケア不足地域 | 幼児教育・医療ケア準備学部 |
| 農林水産業地域 | スマート農業・地域資源活用学部 |
| 観光・文化資源地域 | コンテンツ制作・インバウンド実践学部 |

## 2. 予算を正しく機能させる3つのガバナンス

### ガバナンス①：教員給与は法律に則り、現行水準からの引き上げを義務化

教員給与は公立学校教職員給与法等の法律に則る。ただし**現行水準からの引き上げを義務化**し、上記の水準目標（40歳で上位10%相当）を法令で定める。年功序列の硬直した給与体系は維持せず、実績・評価に連動した柔軟な報酬設計を認める。

### ガバナンス②：公立高校への「個別寄付」の完全禁止とPFS配分

OBの財力や地域の貧富の差が高校間の新たな格差を生まないよう、公立への個別寄付は禁止する。ただしふるさと納税等は公立（コモンズ）のみ受け付ける（私学は不可）。

都道府県による**「3年ごとの成果連動型（PFS）配分」**でイノベーションを起こす学校へ傾斜配分する。

### ガバナンス③：財務・契約・成果の「完全オンライン公開（ダッシュボード化）」

予算の契約履歴・住民投票の結果・執行内容・施設整備の計画をインターネット上でリアルタイム公開する。学校運営への権限委譲と透明性の完全公開をセットにすることで、癒着・無駄遣いを市民の目で監視する。

## 3. 本提言がもたらす3つの「解」

**① 不健全な公費依存からの脱却**

税金で私立を延命させるのを止め、廃校でなく公立化という転換で受け皿を維持したまま公費を集中する。

**② 真の教育の機会均等**

親の財力に関係なく、「地元の公立高校に行けば、優秀な教師のもとで最高水準の教育が完全無料で受けられる」という国家としての本当の安心安全（コモンズ）が完成する。

**③ 地方創生の決定打（10〜20年スパン）**

教育への集中投資を起点に、インフラ・企業誘致・人口流入・税収増という連鎖が自律的に動き出す。補助金で一時的に人を呼ぶ従来策とは根本的に異なる構造転換。

## 4. 前提条件・今後の検討課題

- 移行期間中（最低3年）の私立在校生への支援継続の法的担保
- 公立化した元私立校の教職員・施設・債務の引き継ぎ方法
- 教員給与引き上げの水準を定める法令の設計（年功序列からの脱却と評価連動の整合）
- 過疎地傾斜配分の具体的な算定式（生徒数・人口密度・地域指数等）
- 地域既存施設との重複自動照合システムの設計・データベース整備
- 住民投票システムのセキュリティ設計（マイナンバー連携・なりすまし防止）
- 年齢別加重投票・中高生への投票権付与の法的整備
- 学習指導要領廃止後の都道府県カリキュラムの質担保
- PFS（成果連動型配分）の評価指標の設計
- 施設整備の「意味のある投資」判定基準の明文化
- 公立高校へのふるさと納税活用の法的整備
