# 提言010：土地・不動産の外国人・外国資本取得規制プラン

**～国土は誰のものか。所有権の原則を取り戻す～**

---

## 0. 問題意識

日本は世界でも稀な「不動産取得が完全に自由な国」である。外国籍であろうと外国法人であろうと、日本人と全く同じ条件で土地を購入できる。一方、東南アジア諸国の多くは外国人の土地所有を禁止または厳しく制限しており、日本の制度は国際的に見ても異例に緩い。

この緩さが、以下の3つの問題を引き起こしている。

- **安全保障リスク**：水源地・国境離島・防衛施設周辺の土地が、外国資本によって実質的に支配される事例が報告されている。実効支配が進めば、最終的には国境線そのものに関わるリスクとなる
- **住居費高騰**：都心の高額マンション（億ション）市場では、新築購入者の2〜4割が外国人という調査結果がある。投資マネーの流入が、日本人の住宅取得をますます困難にしている
- **既存法制の機能不全**：2022年施行の重要土地等規制法は、自衛隊基地等の周辺土地の「調査権」を国に与えたが、購入そのものを禁止する権限はない

本提言は、土地の所有権を国籍・出資構造に基づいて厳格に制限し、安全保障と住宅市場の両面から国土を守る制度を提示する。

## 1. 土地（戸建て等）の取得規制

### 取得できる者・できない者

| 区分 | 取得可否 |
|---|---|
| 日本国籍（帰化を含む） | ✅ 可能 |
| 永住者（永住権保持者） | ✅ 可能 |
| 上記以外の外国籍個人（就労・留学・観光等の在留資格） | ❌ 不可（賃貸のみ可） |
| 日本法人（外国資本比率15%以下） | ✅ 可能 |
| 日本法人（外国資本比率15%超） | ❌ 不可 |
| 外国法人 | ❌ 不可 |

永住者は法務大臣による厳格な審査（原則10年以上の在留・素行善良要件等）を経て「国が認めた」存在であるため、帰化者と同等に扱う。それ以外の在留資格保持者は、日本に居住していても土地の購入は認めない。

外国資本15%という基準は、外為法の対内直接投資審査制度等で用いられる警戒ラインを参考に設定した。日本法人を経由した「実質的な外国資本による土地所有」という抜け道を防ぐための基準である。

### 既存所有者への遡及適用

```
制度開始
　↓ 即時カウント開始（告知期間なし）
5年間の猶予期間
　↓
まず国内の民間希望者への売却を試みる
　↓ 買い手が見つからない場合
国が最終的に買取を保証する
```

既存に土地を所有している外国人・外国資本についても、既得権は一切保護しない。**ここは日本の国土であり、日本の国土に関する制度に外国人が異議を唱える立場にはない**という原則に立つ。

5年以内に売却できなかった分は国が買い取るが、実際の運用では「外国企業が利用したい土地は日本企業が買って貸せばビジネスとして成立する」という市場原理が働くため、国の買取保証が実際に発動するケースは限定的と想定する。買取の原資は、後述する外資への賃貸料収入等を充当する。

### 外資の土地利用：購入は不可だが賃貸は可能

一般エリアにおいては、外国企業が工場等を建設する目的で土地を利用すること自体は妨げない。ただし方法は**賃貸に限定**する。

```
外国企業（例：半導体メーカー等）が日本に工場を建てたい
　↓
土地は購入できないため、日本企業（地主）から借りるしかない
　↓
賃貸条件（賃料水準等）は地主である日本企業の裁量に委ねる
```

安価な賃料設定によって誘致を有利に進めることも、経済安全保障上のパートナーシップ強化の「武器」として活用できる。土地の所有権は常に国内に固定されたまま、利用権だけを戦略的に動かす設計である。

## 2. 重要エリア：購入も賃貸も全面禁止

### 対象エリア

- 水源地（ダム・取水場周辺）
- 国境離島
- 防衛施設・自衛隊基地周辺
- 原子力発電所・電波塔等の重要インフラ周辺
- その他、国が指定した区画（将来の新たなリスクに備えた余白条項）

### ルール

重要エリアでは、一般エリアで認められている「賃貸による利用」も含めて**外国人・外国資本への提供を全面禁止**する。理由は、所有権の規制だけでは「賃貸契約による実質的な排他的支配」というグレーゾーンを防げないためである。

### 監視：国による年4回の定期調査

事業者側からの自己申告（報告義務）ではなく、**国が能動的に年4回、現地を調査する**。調査内容は以下を含む。

- 土地の使用状況の確認
- 所有者・実質使用者の特定
- 資金源の追跡

これにより、「日本国籍者が名義人となって購入し、実質的には外国資本が支配する」という**名義貸しによる脱法行為**も同時に摘発する。名義貸し対策を独立した制度として新設するのではなく、年4回の定期調査の中に統合することで、行政コストを抑えつつ実効性を確保する。

## 3. マンション（区分所有）の取得規制

### 戸建てとは異なる設計思想

戸建て等の土地が「国籍・資本構造による規制」であるのに対し、マンションは構造的に外国人個人が敷地全体を実質支配することがあり得ない（あくまで建物の一室＋持分比率の権利）ため、**「誰が買うか」ではなく「どう使うか」を規制する**、用途ベースの設計とする。

| 項目 | 内容 |
|---|---|
| 購入できる人 | 国籍を問わず誰でも可能（戸建てのような国籍制限なし） |
| 購入の条件 | 居住用としての取得に限る。マンションの敷地権も同様の扱いとする |
| 禁止行為 | 購入した上で他者へ賃貸に出す（投資・賃貸目的の所有）は外国人について不可 |

### 居住実績ルール：3年で400日以上

購入時点では誰でも取得できるが、**購入後の継続条件**として居住実績を課す。

```
購入（誰でも可・居住意思があればよい）
　↓
購入後3年間で400日以上の居住実績が必要
　↓
未達の場合：罰金・課徴金を科す（所有そのものの継続は可能）
```

居住実績ルールにより、「購入はしたが実際には住まず、値上がり待ちの投資資産として放置する」というパターンを排除し、実需に基づく購入のみを許容する。

**正当な理由による緩和**：海外赴任等のやむを得ない事情がある場合は、申請により居住日数の緩和を認める。ただし**連続申請は不可**とし、安易な制度の濫用を防ぐ。

### 戸建てとマンションが組み合わさることで生まれる相乗効果

```
土地は外資に売れない（賃貸のみ）
　↓
タワマンを建てたい外資デベロッパーは土地を「借りる」しかない
　↓
借地である以上、地主（日本企業）の方針転換リスク（賃料値上げ・契約打ち切り等）が常につきまとう
　↓
リスクの大きい借地に巨額投資（タワマン建設）するのは敬遠される
　↓
外資による大型タワマン開発そのものが自然に抑制される
　↓
同時に、マンションの居住実績ルールが投資目的の購入も抑制する
```

土地の所有規制とマンションの用途規制が両方揃うことで、「外資マネーによる住居費高騰」という問題全体に対する二重の防波堤として機能する。

## 4. 経済効果の試算

本提言は安全保障・住宅市場の両面でプラスの効果を狙う一方、**新規供給の停滞という経済的なマイナス影響も伴う**。両方を正直に試算する。

### 前提データ

| 指標 | 数値 | 出典 |
|---|---|---|
| 国外居住者の新築マンション取得割合（東京23区・2025年1〜6月） | 3.5% | 国土交通省 |
| 同・都心6区限定 | 7.5% | 国土交通省 |
| 都心3区（千代田・港・渋谷）の高額帯（億ション） | 2〜4割（業界アンケートでは5割超の回答も） | 三菱UFJ信託銀行調査 |
| 都心湾岸タワマン（1億円超）の所有権移転 | 29.9%が外国人 | 同上 |

国土交通省の登記ベース調査（実数に近い）と業界アンケート（高額帯に偏ったサンプル）で数字の開きが大きい。全国・全価格帯では数%、都心高額帯では2〜4割という**二層構造**を前提に試算する。

### マイナス影響：新規供給の停滞

外資デベロッパーが土地を購入できず賃貸でしか確保できなくなることで、方針転換・賃料上昇リスクを嫌気し、大型タワマン開発から撤退する可能性がある。

| シナリオ | 想定 | 影響規模 |
|---|---|---|
| 保守的試算 | 都心高額帯（2億円以上）の外資系開発案件が縮小し、当該セグメントの供給が1〜2割減 | 都心部新築供給全体への影響：約△1.5〜3% |
| 楽観的試算 | 外資撤退は限定的。デベロッパーが日本企業中心へ切り替わり吸収 | 影響はほぼ誤差範囲（△1%未満） |

### プラス影響：価格下押し効果

外資需要（都心高額帯の2〜4割）が市場から退出することで、需給バランスが変化する。

| シナリオ | 想定 | 影響規模 |
|---|---|---|
| 保守的試算 | 高騰は完全には止まらないが、加速が抑制される | 都心高額帯の価格上昇率が年1〜2ポイント程度鈍化 |
| 楽観的試算 | 投資マネー流入が止まり、価格水準そのものが調整される | 都心高額帯で5〜10%程度の価格下落。周辺相場にも一部波及 |

### バランスの総括

| 項目 | 保守的シナリオ | 楽観的シナリオ |
|---|---|---|
| 新規供給への影響 | 都心高額帯で1〜2割減（全体では△1.5〜3%） | ほぼ誤差範囲 |
| 価格への影響 | 都心高額帯の上昇率鈍化（年1〜2pt） | 都心高額帯で5〜10%程度の価格調整 |
| 日本人の購入しやすさ | 限定的に改善（高額帯中心） | 都心高額帯で明確に改善。中間層にも一部波及 |

### 試算の限界（重要な留保）

本試算には大きな不確実性が伴う。第一に、外国人取得割合の公的統計は2025年に整備が始まったばかりで、長期トレンドの裏付けが薄い。第二に、不動産価格高騰の主因が外資需要なのか、それとも低金利・建設費高騰なのかについて、専門家の見解は割れている。第三に、新規供給の減少が「希少性の向上」を通じてかえって価格を押し上げる可能性も理論上は否定できない。

本提言はこれらの不確実性を認めた上で、安全保障上のリスク低減という非経済的な便益と合わせて、総合的に妥当性を判断すべきものである。

## 5. 全体構造のまとめ

| カテゴリ | ルール |
|---|---|
| 戸建て等の土地（一般エリア） | 永住者・帰化者・外国資本15%以下の日本法人のみ取得可。それ以外は禁止（賃貸のみ） |
| 戸建て等の土地（重要エリア） | 購入・賃貸とも全面禁止。国が年4回調査 |
| 既存所有者（土地） | 制度開始から即5年カウント。民間優先売却→国が買取保証 |
| マンション（区分所有・敷地権含む） | 居住目的なら誰でも購入可。投資・賃貸目的は外国人不可 |
| マンション居住実績 | 3年で400日以上。未達は罰金（所有継続可）。海外赴任等は1回限り緩和申請可 |
| 外資の土地利用（一般エリア） | 賃貸は可（半導体工場等の誘致は継続） |

## 6. 前提条件・今後の検討課題

- 「外国資本15%」の算定方法（議決権ベースか出資比率ベースか、間接保有の扱い）
- 永住権取得のハードルと土地取得インセンティブの関係性の検証（永住権取得の濫用リスク）
- 国による買取保証の具体的な予算規模・原資となる賃貸料収入の試算精度
- 重要エリアの「その他国が指定した区画」の指定プロセス・基準の明文化
- 年4回の定期調査を実施する体制・人員・予算の設計
- マンションの居住実績「3年で400日」の確認方法（住民票・電気水道使用量等の組み合わせ）
- 海外赴任等の緩和申請における「正当な理由」の判定基準の明文化
- 既存の外国人所有者（特に投資目的）からの反発・国際的な反応への対応
- 重要土地等規制法など既存法制との整合性・統合の要否
- 外資による工場用地賃貸の賃料設定が国際通商ルール（WTO・FTA等）と抵触しないかの精査
